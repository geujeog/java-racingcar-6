### :pencil2: 기능 목록
- 경주할 자동차 이름 입력
  - 쉼표(,)를 기준으로 이름 구분
  - 이름은 5자 이하로 구성
  - 잘못된 값 입력 시, `IllegalArgumentException` 발생

    
- 시도할 횟수 입력
  - 자동차가 이동하는 횟수와 동일
  - 잘못된 값 입력 시, `IllegalArgumentException` 발생


- 자동차 전진
  - 0 ~ 9 사이의 무작위 값을 구한 후, 해당 값이 4이상일 경우 전진
  - 입력한 자동차 순서대로 전진 상황 출력
  - 한 칸씩 전진


- 게임 완료
  - 한 명 이상의 우승자 출력
  - 우승자가 여러명일 경우, 쉼표(,)를 기준으로 출력
